Uploaded first video on how to install Termux and Git on an Android telephone. The channel is live!

Click here to see the Termux and Git install

This is great as Termux is an Android terminal emulator and Linux environment app that works with no rooting or setup required.

A minimal base system is installed automatically - additional packages are available using the APT package manager.

The idea is to use Termux in the future within my android phone and Chromebook and possibly document findings here.
